Welcome to Epic in C#.

Epic in C# is a translation of the Epic v.0810 software in C#. The goal of this project was to take the original Fortran files of Epic v.0810 and rewrite them in a modern language, C#. Epic in C# was compiled with either Visual Studio or MonoDevelop.

Original Fortran files can be found here.

Information

If you would like to install our software, either click the links to the left or do it manually in a terminal, as shown below.

$ cd your_new_repo_location
$ git clone https://github.com/gk12ksu/Epic-in-CSharp.git
$ cd Epic-in-CSharp
